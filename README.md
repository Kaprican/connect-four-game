# Игра "4 в ряд"
Connect four


## Описание
Веб-версия классической игры "4 в ряд" (Connect Four).
Игра реализована на фронтенде с использованием HTML, CSS, React и TS.

[Поиграть](https://connect-four-game-vercel.vercel.app/)

![Превью игры](public/preview.gif)

## Технические особенности

В игре в качестве системного языка использовала английский — планировала поиграться с интернализацией.
Но времени не хватило, а переписывать все на русский не особо хочется,
поэтому оставила все как есть __┐(￣ヮ￣)┌

Еще добавлю, что для конфетти использовала либу.
В приоритете в данном случае я ставила именно приятные ощущения от игры, поэтому надеюсь на понимание.

### На что обратить внимание
- поддержка системной темы (темная и светлая)
- настроенный пайплайн при push-е
- плавность игрового процесса (звуки, анимация, подсветка)
- демо и документация


## Quick start
1. Склонируй репозиторий любым удобным способом
2. Перейди в папку проекта
2. Установи зависимости через команду `npm i`
   - Важно! для разработки я использовала node v24 
3. Запусти проект через команду `npm run dev`
